系統需求:
Blender 2.8(目前使用) 或更高版本:更高的版本可能因 API 的不同需修改程式內容。



使用方法;
1、打開 Blender 2.8。
2、前往畫面右上 Scripting 標籤，點擊 New 按鈕以建立新的 Python 腳本。
3、複製並貼上腳本內容。
4、點擊 Run Script 按鈕來運行腳本。



程式簡介:
1、Generate_dataset.py : 生成 3D 物件、進行切割、拉伸、並導出模型及深度圖。腳本會生成三個文件夾來儲存不同的輸出內容：3D 物件、深度圖和三角形資料。

功能
一、3D 物件：儲存為 .obj 文件，包含隨機高度的矩形區塊。
二、深度圖：渲染每個模型的深度圖，並儲存為 PNG 格式。
三、角形資料：將物件的三角面轉換為頂點資料，並寫入到指定的檔案中。



2、Restore_3D.py : 從一組深度圖（PNG 格式）生成 3D 模型，並將其導出為 .obj 格式。

功能
一、讀取指定資料夾中的深度圖（PNG 格式）。
二、基於每個深度圖生成 3D 網格，並將其應用位移修改器來模擬蝕刻的 3D 表面。
三"使用相機視角對模型進行 UV 投影。
四、導出生成的 3D 模型為 .obj 格式。
五、支援批量處理資料夾中的所有深度圖，並自動將每個模型儲存為 .obj 檔案。
